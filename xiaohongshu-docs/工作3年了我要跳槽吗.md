# 工作 3 年了，我应该要跳槽吗？

近期粉丝问的比较多的问题是，工作 3 年了，我应该要跳槽吗？有点个人心得，share 下：

1. 第一份工作一定要去找最优秀的团队，一般锚定大厂核心业务和头部独角兽公司即可。

2. 假设已经工作了三年，空杯心态按部就班的学习，几乎都能成长到腾讯 7-8 级，字节 2-1 一年左右。这时候薪酬预估 50-60w/年。这个时候大家想看看机会是特别正常的。 这个时候跳槽是所有公司都最欢迎的，毕竟有其他公司帮你培养了职场的基础，帮助你完成了学生到工作的转身。

3. 如果当前团队 leader 没有画饼，也看不到成长的明确规划，跳槽是职级变现的最优解。2025 年底 xhs 深圳团队就启动了大规模招聘，不少人就实现了职级变现，收入涨幅直接 50%。2025 大疆也有大量岗位招聘。涨幅也在 30% 往上。

4. 我还能去如日中天的业务吗？比如我在 pcg 呆了 3 年，想去 wxg 可以吗？当然可以，不过内部平薪，年终会多几个月，需要大概 5 年左右进入第一梯队，期间晋升和年终不会有太大浮动，除了有镀金作用，收入并没有拉满。

5. 现在 ai 如日中天，对很多年轻人来说是一个弯道超车的机会，现在 ai 岗位的人会比同职级其他岗位多出 30% 的薪酬，如果当前业务有这方面的沉淀，做出成绩后，果断跳槽变现是最优解。
