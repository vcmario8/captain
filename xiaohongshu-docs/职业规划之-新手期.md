# 职业规划之——新手期

最近粉丝问的比较多的是，有两个 offer 怎么选，或者是我该不该离开现有公司接受 offer？这是一个很综合的问题，其实很难直接给答案，每个人的诉求，背景，发展目标都不同。个人的确定性的成长规划一般会分 3 个时期，新手期，成长期，成熟期。要明白每个时期的目标和作用，才能在大选择上不犯迷糊。

本文聊聊新手期。

新手期一般是 0 到 3 年，这个阶段，什么对你来说都是输入，重心是沉淀你的专业能力和职业习惯。刚毕业的前几年，最重要的是沉淀好习惯，学习行业最先进的知识。要找到优秀的团队。因为新手期的学习就是模仿学习。你就是抄方案，抄代码，抄做事方式。既然都是抄，当然要抄最优秀的。

## 1.  如何找到优秀的团队，是不是直接冲大厂就行了？

答案肯定是否定的，优秀的公司，不等于优秀团队。大厂的业务线往往是大而全，各业务也是千差万别的，一定要要识别出优秀团队。其实也不难，只要去核心业务就行了，一般大厂内部优秀的人，第一时间也会往优秀团队转的，整体上核心业务的团队人才密度就高，人才的标准都是高的，这样的团队，各个方面都很值得抄。

## 2.  小公司值得去吗？

大部分小公司都不值得去的，但如果是行业独角兽，那是可以去的，这种公司深度足够，有业界影响力，人才密度自然就高。但要警惕别太小众。

新手期核心就是要找到优秀的人扎堆的地方，抄他们的工作内容，职业习惯就行了。
