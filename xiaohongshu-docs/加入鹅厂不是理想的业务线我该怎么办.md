# 加入鹅厂，不是理想的业务线，我该怎么办？

## 1 加入大厂的目的是什么？
以加入鹅厂为起点，开始规划后面的路。

## 2 大的思路是什么？
借大厂锻炼自己的能力，然后最大化自己收益。
具体需要沉淀什么能力呢？专业能力肯定要，但不是最核心的。大厂已经严重人才饱和了，技术更是严重过剩，你需要的是解决复杂事情的能力。比如视频号的业务指标是 dau 和时长，它的竞品是 xhs，dy，ks，需要抢夺时长，得进一步下钻看年龄分布，性别分布，地域分布等等，然后理解业务的重点方向和规划，在这个背景下，开始看技术有哪些可以去赋能的。
具体如何最大化自己的收益呢？你大概能到 8 级，也就是工作 3 年左右，快速跳槽看机会，让职级变现。

## 3 加入不同的业务线后。我需要怎么做？
加入 csig，腾讯的入场券，进去后可以对腾讯祛魅，同时拥有腾讯光环。公司内资源和文章共享，牛人很多，技术和专业能力这块就是业内顶端。到了 8 级有两个选择，去 wxg 踏实成为后续骨干，wxg 的护城河深，转过去后做牛做马几年进入梯队，几乎就能拿到互联网 top 收入。第二个是跳槽，pdd/xhs/db 等大厂都可以去，30% 涨幅应该没问题，之后的路就是循环，等待下一波机会。
加入 pcg，多了一条路，先晋升，pcg 的业务对应的技术还是能打的，假设晋升到 11 级，去宇宙条拿个 3-1，几乎在 2 年内可以拉平和 wxg 的收入。
加入 teg，技术人的天堂了，收入，技术，成长达到了完美平衡，只需要安心深入发育就行了，多打磨打磨技术。是为数不多值得技术人深耕的地方了。
cdg 有 cft 和 ams。cft 是牌照属于躺着两腿一叉就能赚钱。ams 虽然被宇宙条掏空了，但背靠微信强势拉回了一点局面。整体属于躺平党的首选，这里不会有太大压力，拿着中等的收入。
ieg，赌狗且游戏情怀重的人才推荐，而且只推荐工作室，不建议中台，安心跟随游戏上线就算成功了。这个不算互联网了。
wxg，业务多并且都是如日中天，这里业务成熟，壁垒高，全球都知道好，人才密度自然高到离谱。选择它你能快速收获光环，稳步上升的收入，做事成就感等等。但就得接受它的排资论辈，凭什么既得利益者要和你分蛋糕。也得接受它晋升慢，因为排资论辈，因此晋升属于维持新老平衡的共识，你得和 ld 匹配才能拿到晋升门票。
如果你能接受它的规则，就安心躺着吧。如果有才华，有抱负，来镀金就行。
