# 绩效沟通——如何表达待提升项？

## Q：你主导的项目结果不符合预期，明显是协作方不配合导致，在项目复盘时，我应该怎么聊？
A：站在个人角度来看，归因于内就行了，不要去说其他协作方的问题，聚焦于自己的部分，比如没做好信息沟通、项目同步，不然复盘会就会变甩锅大会；如果你的+1 甚至更高级别 ld 在场，会给双方打上"做不成事""甩锅""不会解决冲突"的标签，得不偿失。站在项目角度来说，聚焦事情的后续发展，而不是人，比如事情到了这一步，碰到了困难有 123；你的解决思路是 123；解决方案是 123，哪怕真的是你的原因，这招亡羊补牢也是非常加分的，事后来看就是"有担当""靠谱"，人非圣贤，孰能无错，从错误中成长、从冲突中成长、从变化中获益才是真修行。

## Q：别人投诉我，投诉的事情和事实明显不符，HR/LD 找我沟通时，我应该怎么聊？
A：肯定是按归因于内的方向去聊，主要讲自己，少讲他人；别人有反馈，肯定是基于特定的事件，最少也是信息没做好对齐，沟通有提升空间；相信职场中大家都不是"没事找事"，有竞争是很正常的，竞争与合作并存才是常态。如果过分去强调对方的问题，容易被贴上"甩锅"标签，未来想撕掉这个标签代价非常大。

归因于内、不付出情绪，是职场牛马的必修课；掌握合理的工作的方案，可以显著减少个人内耗；希望大厂的亦菲彦祖们，早日成长到"成熟"的职场人，早日晋升、加薪。
