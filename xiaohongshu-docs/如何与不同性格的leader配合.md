# 如何与不同性格的 leader 配合？

+1 是职场发展最关键的人，一定要和他形成有效的配合，才能获得更好的绩效、更多的调薪。

+1 是通过其他人拿结果，他对最终结果负责，因此他会关注 2 类问题：1）有什么风险影响结果 ？这是下限，必须要守住，属于温饱问题；2）结果能不能超出预期？这是上限，需要争取，属于发展问题。

所以从项目立项到结束，要随时保持这两点的对齐。先做到"早请示、晚汇报"，磨合一段时间后，就知道哪些事情该同步、什么时机同步了。之后就可以考虑更高阶的技巧，如何与不同性格的 ld 配合？

## 1. 老虎性格（决策型）

优点是目标导向、果断决策、执行力强；缺点是缺乏耐心，容易和别人起冲突。和他配合，就是少找借口，目标驱动；提供丰富、严谨的事实和逻辑，让他去做判断；不要提他做决定，一定要和他对齐目标，不然事后会被叼的飞起。

代表人物是马斯克、任正非，应对之道是论迹不论心。

## 2. 猫头鹰性格（严谨型）

优点是注重细节、追求完美；缺点是随机应变能力差，容易错失决策的最佳时机。大都程序员都是这个性格，能升管理的程序员，都锻炼出了老虎的性格了。想当将军的程序员可以关注下。和猫头鹰型领导相处其实比较简单，讲逻辑、摆事实就行了，如果你是老虎型性格，那就很完美，不停的给决策，他会找出你逻辑的漏洞，帮你完善逻辑，会让你如虎添翼。

代表人物是日本企业高管、唐僧，应对之道是论迹不论心。

## 3. 孔雀性格（表达型）

优点是热情、亲和力强、表达能力强；缺点是专业度差。

程序员群里比较少这种性格的领导，多见于市场、公关；如果猫头鹰成长出孔雀性格，就是营销鬼才了，参考雷总。这种性格的领导其实挺好配合，不要抢风头就行。

代表人物是马云、猪八戒，应对之道是论心不论迹。

## 4. 考拉性格（耐心型）

优点是温和、耐心、回避冲突、韧性高；缺点是主动性差。

其实非常适合 toB 行业和客服；大厂这种性格的领导应该比较少的，如果你是这种性格，那就是无限自我内耗，很累。

代表人物是甘地、沙僧，应对之道是论心不论迹。

## 5. 变色龙性格（复合型）

优点是适应能力强、中庸；缺点是：墙头草、油；

比较适合跨部门协调和沟通的，属于资源整合型人才，很多 HR 是这种性格；和他相处，比较轻松，因为他能适配你的性格，但也是追求面面俱到，所以不容易做出深度。

代表人物是外交官、HR，应对之道论心论迹都可以。
