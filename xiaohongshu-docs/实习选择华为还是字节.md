# 实习选择华为还是字节？

有个同学私信拿了实习的 offer，华为服务端和字节飞书客户端，同学未来从事服务端开发，想问问选择哪个？

## 其实同学已经有了选择

如果是字节的服务端开发就不会纠结了。

## 哪边留下的概率比较大，就选择哪边

有个保底的 offer，校招更从容。

## 如果校招一定要重新找互联网的服务端开发

那么字节的实习经历会更加分。

## 其实外界高估了字节，低估了华为

同样是强调奋斗，华为会更长期主义一些，特别是海思等偏硬件的部门，对经验要求高一些，对应的年龄危机也没互联网强。

## 初期互联网的薪酬比华为会高一些

而工作多年后，华为的收入是会越来越高的，特别是每年的分红。
