# 阻碍成长的工作方式——小作坊式

## Q：什么叫小作坊式干活？
A：最重要的标志就是上不了台面，也就是 ontable 能力弱。比如给的方案文档需要专业清晰，会议结论需要有人/事/时间……清晰直接才是职场本质。小作坊式干活最明显的特征就是群里不敢发言，开会不敢 cue 人，怕得罪这个人/那个人，导致想推进的事情没进展，又通过会后挨个私聊去找补，效率低且内耗严重。

## Q：为什么会有小作坊式干活？
A：正常来说，中国毕业生前三年的职场都是这个阶段，怕 leader 、怕得罪人，需要时刻照顾他人感受……越是有点小聪明的人这方面顾虑就越重，主要也是国内的社会环境导致的，所以也不用太回避这个，大多数人都一样。

## Q：那如何破局呢？
A：首先心态上得正视它，这东西很普遍、也很简单，也是很容易解决。其次得明白职场中大家是协作关系，大家聚在一起为了达成组织的目标，以这个作为切入点即可，其余一切顾虑都是自我内耗，因此，谁阻碍你达成任务目标，那你就得去解决，找你+1、对面+1，把问题明牌清楚，协商合作机制。

## Q：小作坊式最大的危害是什么？
A：就是打不了高端局，如果按 solid 的 2-2/T10 要求来看，ontable 是基线要求，如果晋升到这个职级还不能改掉小作坊式干活，那几乎就是大家所说的混子了，大事干不成，小事效率低。
