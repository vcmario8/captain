# 认识鹅厂番外—TME

## 1. 收入
年终奖大家都看到了，4-6 和 6-8 两个档位，每年几乎差不多。除了 wxg 和 ieg 头部工作室 tme 暴打集团内大多数业务线。从腾讯独立出来后，合并了酷 x 音乐，彻底垄断音乐内容板块，属于护城河极深的业务，前几年股市最差的时候依旧坚挺，是少有的能穿越周期的公司，只能说早期进入一直坚持到现在的人真爽。

## 2. 入职难度
离开了集团后，少了一些光环，入职难度比腾讯内部低多了，但内部技术，以及资料都和集团共享。而且有了 tme 的工作经历后，外部大都比较认可。

## 3. 业务
大概年300 亿左右的营收，业务线有 Q 音，酷 x，K 歌。
Q 音，是 tme 核心本部，当年 ross 亲自带队独立出来的业务，每年也给集团发年会员，整体属于内容驱动运营，运营驱动产品，和 ovbu 类似，内容和运营为王，产品线本身是低人一等的。很多以前 sng 老同事，作风也是纯正的腾讯做派，几个嫡系带着一堆牛马干活，嫡系占了大部分团队资源。因为也没啥突破的空间，谈资论辈应该也存在，大有人以斜躺为主，躺不平装卷，卷给老板看。产品规划和发展以 GM 为主就行，腾讯做产品大都不需要带脑子。
酷 x，被 tme 兼并的业务线，对原来团队的同学属于高攀进鹅厂系了，整体算低人一等，不过按腾系的风格。对外部同学应该还挺 nice，如果是 ali 系，早就被洗牌了。
K 歌，早期在 sng 孵化的明星产品，算是 tme 创新探索的标杆了。最主要的产品功能迭代都在这块，算是 tme 最卷的地方了。
其实 tme 有个非常好的优势被大家忽视，因为有直播，k 歌等音视频行业知识，技术栈完美对接抖音，快手，视频号，有了这个经验，去这些头部短视频问题，非常稳。

## 4. 成长
成长是不可能的，ross 是非常有人情味的 leader，收留了很多以前 sng 的老人，也就意味着个人上升通道彻底堵死。但因为其独特的行业经验技术可以完美复刻到大厂业务线，还是非常推荐的。
嫡系盛行，但嫡系不是贬义，大部分终其一生也达不到嫡系的能力，嫡系之所以是嫡系，是因为他们的能力绝对超出身边的人，哪怕换个老板和业务，他们也是嫡系。
