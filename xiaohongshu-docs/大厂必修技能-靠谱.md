# 大厂必修技能——靠谱

## Q：什么是靠谱？
A：靠谱不是有求必应，而是事事有回应，有反馈。比如开会了一定有结论、todo 一定有跟进、让事情一直向前滚动……靠谱的事情做得多了，就能得到靠谱标签，事情交给你，大家都会放心，这就是靠谱，就像 TCP 协议随时有 ack，靠谱标签需要时间累积，

## Q：为什么要靠谱？
A：大厂工作的本质是协作，进大厂的人有很优秀，优秀的人之间的协作依赖共识，而靠谱就是推动事情往前不停的给出共识，持续的给出共识的结论就是靠谱。
优秀的人工作产出不一定好，但靠谱的人绩效一定好。因为靠谱能持续推进事情，不管是创新/成熟业务，都需要这样标签的人。优秀的人常有，靠谱的人不常有…

## Q：怎么做？
A：不要错过工作中的任何信息，比如群消息、会议结论、邮件…特别是 at 你的消息，正面回应，保持公开可接受。
形成有自己特色的做事习惯，比如收到新任务需要按 smart 来复述、参加的会议有明确的议题和结论、陈述项目按金字塔原则来组织…
总结起来就是，凡事都需要准备，时间和精力不够怎么办？ ——需要合理地分配精力，有聚焦有重点，如果持续忙出天际，大概率是偏离了主线，需要 reset 重新启航。
