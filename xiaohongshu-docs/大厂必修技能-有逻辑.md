# 大厂必修技能——有逻辑

## Q：什么是有逻辑？
A：llm 为了回答你的一个问题，内部会反复推理解释，它为什么给出这个答案，这个过程也叫决策链，llm 的思考过程就叫逻辑。如果去看它的思考过程，你就会相信它的答案，这就是有逻辑的魅力，它能让听众信服。

## Q：为什么需要有逻辑？
A：有三个关乎个人成长的关键场景，会重度考察逻辑性。
第一个，总结汇报工作的场景，如果只是念结果，ld对你的判断就很不好。比如你是一线同学，你 ld 判断就是这个同学做的还不错但停留在知其然不知其所以然，还需要沉淀沉淀。比如你是虚线 leader/owner， 你 ld 的判断就是对整体本质掌握不够，有点像传声筒……没逻辑的同学会上回答问题一般都是说我下来再看看。
第二个，和 ld 争取资源的场景，你有一个项目或者专项需要投入，你需要向 ld 申请资源，你肯定得说明这个事情的投入、产出、优先级、推进节奏…然后找你老板，你老板有几个随时脱口而出的问题：这个事情不做有什么损失？和正在做的事情相比，哪件正在做的事情停下来哪个事情的损失你可以接受？这件事最主要解决的问题是什么？最小投入 mvp 的工作怎样？为什么这么划分？……
第三个，晋升和面试的场景，评委和面试官肯定不了解你事情的背景，但考察的是可迁移能力，因此评委不需要很了解你做事的细节，需要重点考察的是你做这个项目的权衡和思考，这个就叫逻辑。有了这个逻辑，大概率做其他项目也能成功，这个就叫可迁移能力，如果只是念出来 star 结构文字，是体现不了可迁移能力的。

## Q：怎么做？
A：可以总结成一个规律——连环 5 问 why。比如项目有了结果，结果怎么归因的？为什么是这几个归因？这个归因继续往下问三层…
