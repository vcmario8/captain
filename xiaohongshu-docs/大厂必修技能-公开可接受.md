# 大厂必修技能——公开可接受

公开可接受也叫 on table，是一种区别小作坊式工作方式的能力。

## Q: 为什么需要 on table 能力？
A: 大厂协作比较多，分工比较细，分工细了后，很多事情是一个人或者一个团队搞不定的，在大家有各自主线目标的情况下，如何快速达成共识、共识后更高效推动事情就特别重要。on table 是指把信息公开出来后，让大家审核信息的客观性，是一种低成本的协作管理方式。

## Q: 什么是 on table 能力？
A: 把你和协作方达成的结论做公示和留痕，这个过程叫共识。公示是最重要的环节，它会约束出结论的人必须客观、也会约束事项跟进人按照契约交付结果，同时把信息周知给项目的相关方，接受大家的任何质疑和提问，也让大家拿着信息能够群策群力。

## Q: 如何锻炼 on table 能力？
A:  回顾工作的生命周期，会有找事情、立项、方案、推进、总结。每个步骤都会涉及到和多个相关方做大量沟通，例如私聊/开会/白板讨论/私聊……在这几个沟通方式下，都需要沉淀结论，结论三要素包括事项、人、时间，然后把结论在项目群范围公示。只要把这个动作做出来就行了，做出来就叫 on table。其他方式都会导致项目推进受阻，要么有人忘记交付他的结论，要么就是出问题后找不到负责人……
