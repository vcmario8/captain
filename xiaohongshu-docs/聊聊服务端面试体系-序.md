# 聊聊服务端面试体系-序

大厂面试通过率<10%，每次晋升季也经常听到同学们说，不知道怎么提升技术。自己站在服务端视角，整理下以前写过的笔记，做一个服务端知识体系的分享。这篇先给下整体框架。后续每个部分单独展开。

结合图 2 来看，一般服务端比较常见的知识点是这些。

## 项目

- 高并发：流量热点/存储热点/快速失败/降级/...
- 缓存：命中率/构建/添加/删除/...
- 稳定性：重保/MTTx
- 服务重构：领域/无损发布/...
- 存储:mysql/newsql/...
- 消息队列: kafka/rocketmq/rabbitmq
- NoSQL:redis/...
- 数据一致性：事务/最终一致/...

## 基础

- 操作系统：内存/进程/文件系统/...
- 网络：tcp/ip/安全/...
- 编程语言：c/c++/java/golang/...
- 算法：数组/字符串/链表/dp/...
