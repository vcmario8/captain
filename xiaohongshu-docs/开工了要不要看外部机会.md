# 开工了，要不要看外部机会？

有粉丝私信问，年后很多人看机会，我应该去看吗？这里给一些个人观点。

1. 看机会不等于跳槽，每年保持一到两次看外部机会，相当于了解更多 context，在必须要跳槽的时候，能随时做出比较合理的选择，所以鼓励大家去外面看机会的。特别的，今年是 ai 大年，几乎一两个月就有比较大的信息变化，比如元宝，豆包，千问等等，这些都是风向标。

2. 跳槽有很多原因，需要找到最核心的原因，这里有一些误区：
   a. 比如说我要跟对的人，每个团队都有人跟对人，只是你没跟对人而已，这个需要积累信任，除非已经做出过成绩且没被认可，否则就是你没做出成绩而已，这个跳槽也就解决不了的。
   b. 比如说团队有嫡系，我拿不到好考核，需要明确的是嫡系的能力就是强，你大概率能力没人强，然后情商还低，这种情况跳槽也解决不了。

3. 跳槽原因无外乎，不爽和给的少。如果这两个同时满足，应该赶紧跳槽。一次调薪，一次绩效，一次年终，就能看出来。啥也没给，也不画饼，果断走。画了饼，再观察最近的饼能不能吃到，吃不到也果断走。
