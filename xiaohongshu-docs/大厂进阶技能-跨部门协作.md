# 大厂进阶技能——跨部门协作

## Q：为什么要跨部门协作？
A：1）分工是生产力提升的标志，小作坊干活方式往往在抢占先机、探索新方向采用；随着市场成熟，提效成为主旋律，就需要分工更细，流水线化效率最高；同时流水线化对企业来说不可替代性最低，不会因为个人卡住项目；2）分工协作，能达成更大的目标，项目成就的天花板更高；3）个人成长需要，高阶人才/管理人才必须对跨部门协作有经验。

## Q：跨部门协作的难点是什么？
A：总结起来就是目标、分工、效率、风险。1）目标不同，每个团队有各自的目标和方向，如何共识？2）边界不清晰，容易抢活、抢功劳；3）进度容易不受控，效率低下；4）过程中的风险多，非常容易有一些三不管地区跟丢。

## Q：怎么解决？
A：1）求同存异，共识目标是第一位。确定目标包括人，事，时间，事情负责人是谁。
2）确定事情分工和事项和节奏，明确事情边界，约定项目跟进机制/上升机制，采用轮值方式主持会议，每次会议议程开始过 todo，项目进度（重点过风险和需要求助）。
3）项目进程中的所有信息需要公开透明，避免同项目组中，不同团队的信息 gap，而且充分的 context 有助于群策群力。具体做法可以沉淀一份事情的 followup 文档，以及建立核心决策群。关键结论一定要开核心层间的会议达成共识，公示后结论广而告之，避免信息 gap。
4）总结时，多感谢项目各方。
