# 大厂进阶工作技巧——合理的精力分配

## Q：什么是精力分配？
A： 有两重含义，一重是工作时间的分配，另一重是深度思考力的分配。大家每天/每周/每个月/每个季度，都会有重点工作和非重点工作，精力需要随着事项重要程度做出调整。

## Q：为什么需要精力分配？
A：大家的工作内容简单分成两类，一类是职责本身需要的工作，属于维持产出的下限，随着行业成熟和供需关系变化，下限要求会越来越高。另一类是能产出预期的事情，在下限做好的基础上挑战自己的上限。因此个人的精力如何分配就得围绕着事情优先级来，这样就能确保下限不拉跨，重点事项投入主要精力。（有同学困惑，这个 Q 没重点事情怎么办？一两 Q 没重点没关系，如果全年没重点，说明你是团队边缘人了，需要主动思考后找自己 ld 聊聊）

## Q：怎么做？
A：无非就是盘点事情和时间，将事情按照重要程度、紧急程度分成四象限；然后按天/按周将时间按是否连续划分成几段，找出连续的时间（比如上午 10:30 前/中午 12-14/晚上 21 点后）；最后将时间和事情做 match ，找出重要不紧急的事情，将连续的时间优先分配给它。
特别说下，能决定上限的事情往往就是重要不紧急的，如果既重要又紧急，一般都是救火工作，或者是季度重点项目，季度重点的话，往往 ld 都会保障时间，精力分配的工作主要是 ld 来做了。但，不管是个人还是团队，如果一直在处理既重要又紧急的事情，是不正常的，这意味着团队/个人一直在扮演消防员的角色，一定是日常工作流程有比较大的缺陷。
