# 认识腾讯—CSIG

## 1 收入
和 PCG 类似，属于烂兄烂弟，今年头部是不错的，有腾讯会议和元宝。能到 8-10 个月，但大多数人就一般般。大多数人 3-4 个月。

## 2 入职难度
最低，属于最水腾讯人了。内部活水也属于鄙视链最低端，pcg 好歹还有老派有名的业务，技术也算不错，csig 是纯大杂烩，有销售，对公商务，官网的 API boy，底层能力包装员……
但有个好处，因为是 toB，需要深入理解不同行业，反而不属于传统互联网，对年龄的容忍度更高一些。

## 3 业务
元宝，腾讯会议，云 123456 部，客服等，还有各种对公的业务。
元宝最近用红包买量的举动让大多数人感受到了当年微视的无奈。微信还是保持一贯的看热闹，就像在说：没我下场不行吧。 整体来说属于没找到方向，字节去年就开始举全公司之力大力发展 ai，豆包已经成长为全民应用了，腾讯落后了整整一年。今年更是勇攀高峰，高峰不常有，同时提升全员收入以提升人才密度，羡慕宇宙条。
腾讯会议，wang 带着以前做 qzone 的同学在疫情期间搞起来的，是 csig 为数不多的技术担当了，今年年终算是支棱起来了，常年 007 的工作吧。总监贼卷，技术也确实牛。
官网和 teg 能力包装 boy，这就就没啥好说的了，未来铁定是子公司化了。腾智欢迎您，本部员工就是带着一堆内包干活。
对公的业务，就是一堆智慧城市的部门。这个已经完全超出互联网范围，就是普通的甲方乙方，立正听讲就行，这是销售的天下。
早些年，为了抢占市场，大力扩张去一个一个的服务中小企业，差点把自己累死，被叫停后今年可算扭亏为盈了。

## 4 成长
这里就是大杂烩，除了元宝和腾讯会议，其他的业务就不建议去了，未来内包化。腾智欢迎您。但确实对年龄容忍度高一些。
