# 闲聊腾讯的薪酬机制

## Q：腾讯base真的低吗？
A： 真的低，腾讯大把 base 低于 50k 的 11 级（近几年把服务奖拆分到 base 中，涨了一些，但还是低），全年 pgk 中，年终奖+ RSU 占了大半。

## Q：什么职级开始有RSU？
A：RSU 和职级没什么特殊关系，有些 11 级也没有；有些校招也有；一般来说和部门有关，wxg 的同学部分年终奖会折算成 RSU；一般一年有一次增发 RSU（社招以合同上的金额一次性授予，合同到期后视情况而定是否增发，这点很坑）；被系统标记为高潜同学都有可能增发，一般就是嫡系，要获得+2 也就是总监的认可，GM 分包后，总监可以微调（拆东墙补西墙，你多了别人就少了）。

## Q：每年 RSU 有多少？
A：每个职级不同，不过对于一线牛马 40w 以上就是很多的那一档了，还有 15w/25w这两档；按当年的某一天的价值授予，分 4 年归属；如果是 leader，那就50w 起了；总监起开始有期权；期权类似二级市场期货，上升期的话赚麻了。需要注意的是，如果离职，已经归属的不受影响，未归属的清零。

## Q：年终奖差异能有多大？
A：腾讯收入是按照 base+年终+当年归属 RSU，归属 RSU 又因当年价格而波动，因此同职级的每个同学的收入差距很大；单看年终奖，BG/部门/团队差异很大；wxg最高（老牌 IEG，王者/吃鸡/穿越火线等头部也很高），wxg 年终会折半到 RSU，因此哪怕毕业生，20w 年终也是很 easy 的；比较稳的是 CDG 支付线和 TEG，略高于 offer 上的月份数；难兄难弟就是 pcg 和 csig，考核达标的情况下，勉强达到 offer 月数，outstanding 绩效和 wxg 的普通一样，但 wxg 有 rsu 加持。当年名品堂产品的话，年终奖会有独立的奖金包，对比往年直接翻倍也是可能的。年终奖是拉开差距的大头。
