# 晋升面试，为什么评委喜欢问一些通用问题？

## 晋升答辩评委为什么喜欢问通用问题？

晋升和面试，其实是类似的场景，是一场说服评委的过程，这个过程中，评委需要考察出候选人的思考深度，然后给出是否晋升的判断。
评委的出发点是判断你能否通过面试，因此他的面评也必须是 solid 的，那你应该有亮点让评委抓到；否则，就算他想让你通过，评语也写不出来；既然写不出来，那就索性不通过了。

## 评委喜欢问通用问题的考虑点

评委什么喜欢问一些通用的问题，一般有下面 3 个考虑点：
1. 业务非常多，评委也不一定全部都了解，因此基于双方的交集展开考察就是一个解决方案。
2. 底层的技术原理是一致的，比如 cpu的 cacheline-内存-磁盘的一致性问题，在很多分布式场景下就变成了数据一致性问题。
3. 每个专业到高阶等级时，底层的思考逻辑是类似的，比如高手总是善于打比方；因此把事情讲明白，也是晋升时考察的子项。
