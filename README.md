## 核心洞察

### 1. 学习阶段特点
- **文化适应阶段**（大厂文化）
  - 重点：行业趋势、职场选择、公司对比、管理文化
- **入门求职阶段**（大厂面试）
  - 重点：面试准备、公司面试流程、offer选择、实习留用、校招求职
- **职业发展阶段**（职业规划）
  - 重点：职业阶段、owner养成、晋升转型、晨会管理
- **技能提升阶段**（大厂进阶技能）
  - 重点：核心工作方法、进阶工作技巧、协作沟通、成长避坑
- **晋升成长阶段**（大厂晋升）
  - 重点：晋升准备、晋升答辩、晋升技巧、职级体系、晋升理念
- **薪酬体系阶段**（大厂薪酬与绩效）
  - 重点：绩效管理、薪酬机制、年终奖、宇宙厂薪酬
- **技术深度阶段**（后端技术体系与架构）
  - 重点：面试体系、系统设计、数据库技术、大厂八股、存储引擎、核心原理
- **前沿趋势阶段**（大厂 AI 进行时）
  - 重点：AI offer选择、AI产品、AI技术
- **公司认知阶段**（认识鹅厂）
  - 重点：业务部门、鹅厂文化

## 使用建议

1. **按阶段学习**：根据当前职业阶段选择对应分类
2. **循序渐进**：建议按从前到深的顺序系统性学习
3. **重点突破**：针对薄弱环节重点学习相关文档
4. **持续关注**：AI相关文档值得持续关注
5. **组合学习**：可将不同阶段的文档组合学习，形成完整知识体系
