# 船长：大厂成长指北

## 内容概览

本指南整合了 120+ 篇关于大厂职场、面试、晋升、技术体系等方面的深度文章，涵盖了从职场新人到技术专家的全方位成长路径。

### 核心内容板块

```
大厂成长指北
├── 大厂文化与职场认知
├── 面试求职与 Offer 选择
├── 职业规划与发展路径
├── 核心工作技能与方法论
├── 晋升体系与职业跃迁
├── 薪酬绩效与激励机制
├── 后端技术体系与架构
├── AI 前沿趋势与选择
└── 腾讯（鹅厂）深度解析
```

### 板块内容说明

- **大厂文化与职场认知**：了解行业趋势、公司文化、职场选择策略
- **面试求职与 Offer 选择**：掌握面试技巧、了解公司面试流程、做好职业选择
- **职业规划与发展路径**：从新手期到成长期，明确不同阶段的发展目标
- **核心工作技能与方法论**：提升工作效率、沟通能力、团队协作技巧
- **晋升体系与职业跃迁**：熟悉晋升规则、准备晋升答辩、实现职业突破
- **薪酬绩效与激励机制**：了解薪酬结构、绩效评估、年终奖分配逻辑
- **后端技术体系与架构**：深入学习服务端面试体系、系统设计、数据库技术
- **AI 前沿趋势与选择**：把握 AI 时代的职业机会、技术发展方向
- **腾讯（鹅厂）深度解析**：了解腾讯各业务部门、文化特点、技术生态
